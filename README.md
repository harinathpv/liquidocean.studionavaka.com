# liquidocean.studionavaka.com
The living companion site to **The Liquid Ocean Compass** by Harinath Pudipeddi.

Next.js 14 (App Router). Design system: "The Imprint" (warm paper, institutional green accent).
Images live in `public/images/`. Deploys on Vercel — set Framework Preset to **Next.js**.

## Develop
    npm install
    npm run dev
